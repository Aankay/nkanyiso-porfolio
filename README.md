# Nkanyiso Mavuso Portfolio

A clean, modern personal portfolio website showcasing the skills, experience, and projects of Nkanyiso Mavuso, a Multimedia Computing graduate focused on software development, 3D environments, and immersive digital experiences.

## Project Overview

This portfolio is built with a lightweight, responsive HTML/CSS design and uses Tailwind CSS for layout and styling. It includes sections for:

- About
- Projects
- Skills
- Experience
- Contact
- Downloadable CV

## Built With

- HTML5
- CSS3
- Tailwind CSS
- Google Fonts

## Features

- Responsive layout for desktop and mobile
- Animated entrance and interaction effects
- Custom cursor and scroll progress indicators
- Download CV button
- Mobile navigation menu
- Contact links for email, phone, and LinkedIn

## Usage

1. Open `index.html` in a web browser.
2. Explore the portfolio sections using the navigation links.
3. Click the CV button to download the resume.

## File Structure

- `index.html` — Main portfolio page
- `index.css` — Custom site styles
 
## Notes

This portfolio is designed to present a professional personal brand with immersive visuals and smooth navigation. It is ideal for showcasing multimedia development and programming capabilities.
